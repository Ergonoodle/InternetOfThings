Plotter running Windows 10 IoT Core
=========================

<img src="https://github.com/cazacov/InternetOfThings/blob/master/PlotterWin10/img/plotter.jpg?raw=true" alt="Plotter running Windows 10 IoT" width="720" height="480"/>

## Parts list ##
- Raspberry Pi 2
- Two old DVD drives
- [Adafruit DC & Stepper Motor HAT](https://learn.adafruit.com/adafruit-dc-and-stepper-motor-hat-for-raspberry-pi)
- Micro servo motor
- [Adafruit I2C PWM/Servo Driver](https://www.adafruit.com/products/815#technical-details-anchor) (optional) to control the servo motor
- Pens
- Stickers
- [Windows 10 IoT Core](http://windowsondevices.com)
